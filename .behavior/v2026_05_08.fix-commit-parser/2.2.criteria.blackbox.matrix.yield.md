# blackbox criteria matrix: cli binary for commit

## matrix.1 = success cases (git repo valid)

| ind: ref type | ind: worktree | dep: stdout gitref | dep: stdout suffix | dep: exit |
|---------------|---------------|--------------------|--------------------|-----------|
| tag at HEAD | clean | tag (e.g., `v1.2.3`) | none | 0 |
| tag at HEAD | dirty | tag (e.g., `v1.2.3`) | `+` | 0 |
| branch (no tag) | clean | branch (e.g., `main`) | none | 0 |
| branch (no tag) | dirty | branch (e.g., `main`) | `+` | 0 |
| detached HEAD (no tag) | clean | commit hash | none | 0 |
| detached HEAD (no tag) | dirty | commit hash | `+` | 0 |

**note**: tag takes precedence over branch when both are present at HEAD
