## pattern 3: commit slug format [REUSE]

**file**: `src/domain.operations/parsers/commit/getEnvCommitFromGit.ts`

**citation [4]** (lines 68-82):
```ts
// prefer tag over branch
const ref = tag || branch;
if (!ref) return null;

// get short hash
const hash = await runGit('git rev-parse --short HEAD 2>/dev/null');
if (!hash) return null;

// check if dirty
const status = await runGit('git status --porcelain 2>/dev/null');
const dirty = status && status.length > 0;

// format: ref@hash or ref@hash+
const slug = `${ref}@${hash}${dirty ? '+' : ''}`;
```

**relation to wish**: sync variant produces identical output format.

**action**: [REUSE] — same logic in sync variant, just with sync git calls.
