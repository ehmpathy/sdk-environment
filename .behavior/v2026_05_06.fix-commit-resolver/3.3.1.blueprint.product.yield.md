# blueprint: sync + async commit resolver

## summary

add sync and async variants to the commit resolver factory `fromGit`:
- `fromGit()` — async (unchanged, backwards compatible)
- `fromGit.sync()` — sync variant via `execSync`
- `fromGit.async()` — explicit async for symmetry

include `getEnvCommit.fromGit.sync` in default sync parsers for `getEnvironment.static()`.

## implementation notes

### error helper extraction

extract shared error check to enable reuse:

```ts
const isExpectedGitError = (error: unknown): boolean => {
  // inline error extraction (matches extant pattern in getEnvCommitFromGit.ts)
  const errorMessage =
    error instanceof Error
      ? error.message
      : typeof error === 'object' &&
          error !== null &&
          'message' in error &&
          typeof (error as { message: unknown }).message === 'string'
        ? (error as { message: string }).message
        : String(error);

  const message = errorMessage.toLowerCase();
  const isGitError =
    message.includes('not a git repository') ||
    message.includes('fatal:') ||
    message.includes('command not found') ||
    message.includes('enoent');
  const isCommandFailure =
    message.includes('command failed') && message.includes('git');
  return isGitError || isCommandFailure;
};
```

### factory extension pattern

extend `fromGit` with static methods. `fromGit()` is a backwards-compat alias to `.async()`:

```ts
const fromGitAsync = (): (() => Promise<EnvironmentCommitSlug | null>) => {
  const parser = (): Promise<EnvironmentCommitSlug | null> =>
    getEnvCommitFromGit();
  Object.defineProperty(parser, 'name', { value: 'fromGit.async' });
  return parser;
};

const fromGitSync = (): (() => EnvironmentCommitSlug | null) => {
  const parser = (): EnvironmentCommitSlug | null =>
    getEnvCommitFromGitSync();
  Object.defineProperty(parser, 'name', { value: 'fromGit.sync' });
  return parser;
};

// fromGit() is backwards-compat alias to .async()
const fromGitFactory = fromGitAsync as typeof fromGitAsync & {
  sync: typeof fromGitSync;
  async: typeof fromGitAsync;
};
fromGitFactory.sync = fromGitSync;
fromGitFactory.async = fromGitAsync;

export const fromGit = fromGitFactory;
```

### sync exec pattern

sync variant uses `execSync` with same commands:

```ts
const runGitSync = (command: string): string | null => {
  try {
    const stdout = execSync(command, { encoding: 'utf8' });
    return stdout.trim();
  } catch (error) {
    if (isExpectedGitError(error)) return null;
    throw error;
  }
};
```
