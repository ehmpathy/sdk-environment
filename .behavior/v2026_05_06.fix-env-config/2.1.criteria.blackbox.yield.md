# blackbox criteria: config separate from access

## usecase.1 = config defaults to access

```
given(access = 'prep')
given(no CONFIG envar)
given(NODE_ENV != 'test')
  when(getEnvironment is called)
    then(environment.config = 'prep')
      sothat(config defaults to access when no signals)
```

```
given(access = 'prod')
  when(getEnvironment is called)
    then(environment.config starts with 'prod')
      sothat(prod access forces prod config)
```

```
given(access = 'test')
  when(getEnvironment is called)
    then(environment.config starts with 'test')
      sothat(test access forces test config)
```

## usecase.2 = prod access forces prod config

```
given(access = 'prod')
given(CONFIG = 'test')
  when(getEnvironment is called)
    then(error is thrown)
      sothat(prod is non-negotiable, invalid override fails fast)
```

```
given(access = 'prod')
given(CONFIG = 'prep')
  when(getEnvironment is called)
    then(error is thrown)
      sothat(prod is non-negotiable, invalid override fails fast)
```

```
given(access = 'prod')
given(NODE_ENV = 'test')
  when(getEnvironment is called)
    then(environment.config starts with 'prod')
      sothat(NODE_ENV does not override prod access)
```

## usecase.6 = CONFIG beats NODE_ENV

```
given(access = 'prep')
given(NODE_ENV = 'test')
given(CONFIG = 'prep')
  when(getEnvironment is called)
    then(environment.config = 'prep')
      sothat(explicit CONFIG beats NODE_ENV signal)
```

```
given(access = 'prep')
given(NODE_ENV = 'development')
given(CONFIG = 'test')
  when(getEnvironment is called)
    then(environment.config = 'test')
      sothat(explicit CONFIG works regardless of NODE_ENV)
```

## usecase.7 = config slug type

```
given(CONFIG = 'prod')
  when(getEnvironment is called with prod access)
    then(environment.config = 'prod')
      sothat(simple tier value is valid slug)
```

```
given(CONFIG = 'prod:v2023_08_21')
  when(getEnvironment is called with prod access)
    then(environment.config = 'prod:v2023_08_21')
      sothat(extended slug format is supported)
```

```
given(CONFIG = 'invalid')
  when(getEnvironment is called)
    then(error is thrown)
      sothat(config slug must start with valid tier)
```

```
given(CONFIG = 'production')
  when(getEnvironment is called)
    then(error is thrown)
      sothat(config slug must start with exact tier value)
```

## usecase.8 = no access detected

```
given(no access can be derived)
  when(getEnvironment is called)
    then(error is thrown)
      sothat(fail fast when access unknown, don't guess config)
```
