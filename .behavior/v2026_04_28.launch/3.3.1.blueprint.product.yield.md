# blueprint: sdk-environment

## summary

implement sdk-environment package to parse and expose runtime environment via:
- `getEnvironment.filled()` — async, runs all parsers, cached
- `getEnvironment.static()` — sync, skips async parsers, cached
- pluggable parsers with first-wins precedence
- typed Environment: `{ access, server, commit }`

## codepath tree

```
getEnvironment.filled()
├── [+] check cache → return if populated
├── [+] run parser chain (all parsers)
│   ├── [+] access: first-wins from getEnvAccess.[fromEnvar, fromAwsAccountAlias, fromNodeEnv]
│   ├── [+] server: first-wins from getEnvServer.[fromEnvar, fromLambdaTaskRoot, fromCiEnvar, fromUnixDesktop]
│   └── [+] commit: first-wins from getEnvCommit.[fromEnvar, fromGit]
├── [+] validate results
│   ├── [+] isEnvironmentAccessTier(access) → throw if invalid
│   ├── [+] isEnvironmentServerTier(server) → throw if invalid
│   └── [+] isEnvironmentCommitSlug(commit) → throw if invalid
├── [+] cache result
└── [+] return Environment

getEnvironment.static()
├── [+] check cache → return if populated
├── [+] run parser chain (sync parsers only)
│   ├── [+] access: skip async parsers
│   ├── [+] server: skip async parsers
│   └── [+] commit: skip async parsers
├── [+] validate results
├── [+] cache result (independent from filled cache)
└── [+] return Environment

parsers/access/
├── [+] getEnvAccessFromEnvar()             # sync: reads ACCESS
├── [+] getEnvAccessFromAwsAccountAlias({ map? })  # async: exact + glob patterns
│       # default map: { '*-prod': 'prod', '*-prep': 'prep', '*-test': 'test' }
└── [+] getEnvAccessFromNodeEnv()           # sync

parsers/server/
├── [+] getEnvServerFromEnvar()             # sync: reads SERVER
├── [+] getEnvServerFromLambdaTaskRoot()    # sync
├── [+] getEnvServerFromCiEnvar()           # sync
└── [+] getEnvServerFromUnixDesktop()       # sync: XDG_SESSION_TYPE | TERM_PROGRAM → 'local@unix'

parsers/commit/
├── [+] getEnvCommitFromEnvar()             # sync: reads COMMIT
└── [+] getEnvCommitFromGit()               # async (shell exec)
    ├── [+] git describe --tags --always
    ├── [+] git rev-parse --short HEAD
    └── [+] git status --porcelain → dirty flag
```
